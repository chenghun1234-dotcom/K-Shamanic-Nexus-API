# K-Shamanic Nexus API 📽️

A high-performance, zero-cost backend engine providing structured JSON data on Korean Shamanism, Folklore, and Occult Mythology. Designed for game developers, writers, and digital creators looking for authentic K-Occult content.

## 🌟 Key Features
- **Folklore Dictionary**: 10,000+ data points on Korean spirits (Dokkaebi, Gumiho, Jangsanbum).
- **Ritual Engine**: Authentic steps and ingredients for shamanic exorcism.
- **Scenario Generator**: Dynamic horror plot generation based on Korean rural settings.
- **Divination Matching**: Saju-based occult matching logic.

## 🚀 Quick Start
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the server:
   ```bash
   python main.py
   ```
3. Access Documentation:
   Navigate to `http://localhost:8000` for the premium UI.

## 🛠️ Technology Stack
- **Core**: FastAPI
- **Data**: Static JSON Database (Zero Operating Cost)
- **UI**: Vanilla HTML/CSS with Glassmorphism Design

## 📍 API Endpoints
- `GET /entities`: Retrieve folklore entity dictionary.
- `GET /rituals`: Get exorcism and ritual data.
- `GET /divination-logic`: Saju matching engine.
- `POST /plot-generator`: Generate occult scenarios.

## 💰 Monetization
Ready for deployment on **RapidAPI**. Uses an OpenAPI schema for easy integration.
